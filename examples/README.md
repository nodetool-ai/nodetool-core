# NodeTool Core Examples

Examples demonstrating Python node execution.

## Examples

- `simple_node_execution.py` — Basic node creation and execution with ProcessingContext
- `asset_tracking_example.py` — Asset storage and tracking
- `audio_stream_example.py` — Audio streaming with numpy arrays
- `model3d_example.py` — 3D model asset handling
- `minimax_example.py` — MiniMax API integration

## Prerequisites

```bash
conda activate nodetool
uv sync
```
